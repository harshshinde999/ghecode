PICT B.E. Course Structure

First Year (FE)

  • Engineering Mathematics I
    Fundamentals of calculus and linear algebra.
  • Basic Electronics
    Introduction to electronic components and circuits.
  • Programming in C
    Basics of programming using C language.
  • Engineering Mechanics
    Statical and dynamical analysis of mechanical systems.

Second Year (SE)

  • Data Structures
    Study of stacks, queues, trees, graphs.
  • Computer Organization
    Understanding the architecture of computer systems.
  • Discrete Mathematics
    Logic, set theory, and combinatorics for CS.
  • OOP using Java
    Object-oriented concepts implemented in Java.

Third Year (TE)

  • Operating Systems
    Concepts of process, memory, file and device management.
  • Computer Networks
    Study of data communication and protocols.
  • Database Management Systems
    Relational databases, SQL, normalization.
  • Software Engineering
    Software development lifecycle and project management.

Final Year (BE)

  • Artificial Intelligence
    Introduction to AI concepts and algorithms.
  • Machine Learning
    Supervised and unsupervised learning techniques.
  • Cloud Computing
    Cloud architecture, services, and virtualization.
  • Project Work
    Real-world project applying all learned concepts.
